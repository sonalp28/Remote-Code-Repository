When you run this project, all the requirements are demonstrated by automatic process. If you want to access the GUI and check the functionality by yourself, please go the MainWindow.xaml.cs and follow the comments written in the 'Window_Loaded' method.

To run the project -
Method 1 -
1. Open the solution file in Microsoft visual studio 2017.
2. Build the project
3. Run the project

Method 2 -
1. Open windows command prompt.
2. Run the command to go to the path where the project's solution file is located [e.g. cd C:\Users\....
3. Now type compile and hit the Enter [running the compile.bat file]
4. Now type run and hit the Enter [running the run.bat file]
